## 评审视角

### 该专家特有的评审角度

1. **评估框架系统性**: 关注评估框架是否覆盖模型能力的多个维度
2. **方法论可重复性**: 关注实验设计是否足够详细，他人能否复现结果
3. **统计严谨性**: 关注是否进行适当的统计检验，结论是否有足够数据支持
4. **评估基准公平性**: 关注不同模型是否在同等条件下评估

### 重点关注方面

| 评审项目 | 具体关注点 |
|----------|-----------|
| **实验设计** | 控制变量、随机化、样本量计算 |
| **评估指标** | 指标选择的合理性、是否标准化 |
| **结果分析** | 统计显著性、效应量、置信区间 |
| **对比基线** | 基线选择是否合理、是否包含SOTA |
| **可复现性** | 代码/数据是否公开、协议是否详细 |
